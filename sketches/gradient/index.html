Gradient Drift builds a palette from layered fbm noise, then feeds the previous frame through a small rotating offset to create slow color trails.

What is happening

The base image is a smooth noise-selected palette. The feedback lookup shifts by a tiny direction derived from n1n_1n1​, so color boundaries drift without becoming a full fluid simulation.

Key variables

Noise frequencies set the size of color regions, feedback decay (0.95)(0.95)(0.95) sets trail persistence, and the 0.150.150.15 blend controls how quickly new palette color replaces old state.
